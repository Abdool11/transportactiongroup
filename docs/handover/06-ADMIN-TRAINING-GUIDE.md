# TAG Ecosystem — Admin Training Guide

This guide provides step-by-step instructions for the most common administrative tasks across the three platforms. Use this to train the respective platform administrators.

## 1. Green Freight Academy (GFA) Admin

The GFA Admin manages B2B relationships, approves training cohorts, and handles industry-wide communications.

### Task: Approving a Driver Cohort
When a company deploys a list of drivers, it requires admin approval before magic links are sent.
1. Log into the GFA Admin portal.
2. Navigate to **Cohorts** in the sidebar.
3. Locate the cohort marked as "Pending Approval".
4. Review the driver list for accuracy.
5. Click **Approve**. This action automatically dispatches WhatsApp/Email magic links to all drivers in the cohort.

### Task: Broadcasting a Campaign
Campaigns are used to send marketing or informational messages to prospect leads.
1. Navigate to **Campaigns**.
2. Click **New Campaign**.
3. Select the target audience (e.g., all leads, specific tags).
4. Choose the delivery method (Email, WhatsApp, or Both).
5. Compose the message. If using WhatsApp, select an approved Meta template.
6. Click **Send Campaign**.

### Task: Managing Trial Vouchers
Vouchers allow companies to test the platform with a limited number of drivers.
1. Navigate to **Vouchers**.
2. Click **Generate Voucher**.
3. Set the limit (e.g., 5 drivers) and expiration date.
4. Provide the generated voucher code to the prospect company.

## 2. BetterDriver (BD) Admin

The BD Admin manages the B2C driver experience, monitors progress, and handles support.

### Task: Monitoring Driver Progress
1. Log into the BD Admin portal.
2. Navigate to **Drivers**.
3. Search for a specific driver by name or ID number.
4. View their detailed profile to see module completion status, last login date, and certificate status.

### Task: Publishing an Industry Bulletin
Bulletins are important notices displayed in the driver portal.
1. Navigate to **Bulletins**.
2. Click **Create Bulletin**.
3. Enter the title, content, and select a category (e.g., Safety, Route Update).
4. Choose the urgency level. High urgency bulletins may trigger a push notification.
5. Click **Publish**.

### Task: Managing CPD (Continuous Professional Development)
1. Navigate to **CPD Library**.
2. Upload new training materials or link to new Moodle short courses.
3. To push a CPD requirement to drivers, select the item and click **Assign to Drivers**.

## 3. Transport Action Group (TAG) Admin

The TAG Admin manages public enquiries and TCO calculator submissions.

### Task: Reviewing Contact Enquiries
1. Log into the TAG Admin portal.
2. Navigate to **Enquiries**.
3. View the list of submissions from the public contact form.
4. Click on an enquiry to view the full message and contact details.

### Task: Analyzing TCO Submissions
1. Navigate to **TCO Submissions**.
2. View the data submitted by companies using the Total Cost of Ownership calculator.
3. Use this data to identify potential leads for GFA outreach.
